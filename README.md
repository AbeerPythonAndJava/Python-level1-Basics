Smart Autism Assessment System (SAAS)
Building the "Logical Brain" of an AI-driven educational tool for children with autism.

📌 Project Vision
This project is being developed stage-by-stage to create an Adaptive Assessment System. The goal is to monitor child performance and automatically adjust the learning level to match their individual needs.

🛠 Phase 1: Building the Logical Brain (Current Phase)
In this phase, I am focusing on programming the Decision-Making Engine using Python fundamentals.

📅 Week 1: Sensing & Evaluation
Concepts: Inputs, Outputs, and Conditional Logic (if/else).

What I built:

01_session_tracker.py: Calculates session duration (The robot's sense of time).

02_attention_evaluator.py: Classifies attention levels (The robot's ability to judge performance).

📅 Week 2: Memory & Data Management
Concepts: Lists, len(), sum(), and append().

The Goal: Teaching the robot how to "remember" previous sessions and calculate the average progress.

Project: 03_session_log_manager.py (Completed ✅).

Functional Logic: Stores multiple attention scores and provides a performance summary.

🚀 Future Roadmap
Phase 2: Interactive Question Bank (Using Loops & Dictionaries).

Phase 3: User Interface (Web-based interaction for children).

Phase 4: AI Adaptation (Machine Learning for personalized paths).
